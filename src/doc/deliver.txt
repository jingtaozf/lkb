To create a version of the LKB for delivery:

1. Source files.
  a) make sure everything is committed to cvs
  b) create a new directory, e.g. lkbdeliver/vers5-0
  c) cd to this
  d) cvs export -D 12/9/98 lkb (this doesn't give the tsdb stuff)
  e) /bin/rm -r newlkb/src/non-core
  f) tar cf ../lkbsrc5-0.tar .
  g) gzip ../lkbsrc5-0.tar
  h) mv ../lkbsrc5-0.tar.gz ~aac/WWW

2. Image
  a) start Lisp 
  b) (load "general/loadup")
  c) (load "ACL_specific/deliver.lsp")
  d) cd newlkb/bin
  e) tar cf ~aac/lkbdeliver/vers5-0/climimage5-0.tar lkb
  f) gzip ~aac/lkbdeliver/vers5-0/climimage5-0.tar
  g) cp ~aac/lkbdeliver/vers5-0/climimage5-0.tar.gz ~aac/WWW

3. data files
  a) as source to e)
  b) cd lkbdeliver/vers5-0/newlkb/src
  c) tar cf ~aac/lkbdeliver/vers5-0/data5-0.tar data
  d) gzip ~aac/lkbdeliver/vers5-0/data5-0.tar
  e) mv ~aac/lkbdeliver/vers5-0/data5-0.tar.gz ~aac/WWW

4. LinGO grammar
  a) mkdir lkbdeliver/grammar/jun99
  b) cd lkbdeliver/grammar/jun99
  c) cvs export -D 7/16/99 grammar
  d) tar cf ../grammarjun99.tar .
  e) gzip ../grammarjun99.tar
  f) mv ../grammarjun99.tar.gz ~aac/WWW
